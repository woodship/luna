#参与开发方法

1.注册github帐号并登录

2.点击本页右上角的fork，创建一个分支到自己的帐户下

3.clone自己帐户下的分支到本地，进行修改，提交

4.在自己帐户点pull request提交自己的修改申请

5.管理员审核通过后会合并到源代码中

6.如何保持自己的fork与原始库的同步：再建立一个指向原始库的remote,然后fetch,merge
